Exploring deep concepts in async/await behavior to prevent deadlocks in enterprise web APIs.

Focus: Understanding SynchronizationContext and why `ConfigureAwait(false)` is crucial in library code but shouldn't always be used in UI layers.

    // Avoid deadlocks by leaving the synchronization context
    return await _repo.FetchAsync().ConfigureAwait(false);
}
